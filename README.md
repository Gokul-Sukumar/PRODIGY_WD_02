# ecommerce-platform

A modern, fast, and scalable e-commerce platform frontend built with **React 19** and bundled using **Vite**.

## 🚀 Features

- **React 19**: Leverages the latest features and optimizations of the React ecosystem.
- **Vite Production Bundler**: Blazing fast development server and optimized production builds.
- **ESLint 10**: Pre-configured flat config system for modern JavaScript/React linting.

## 🛠️ Prerequisites

Ensure you have **Node.js** (v20.19.0 or higher recommended based on tool specifications) installed on your machine.

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone <your-repository-url>
cd ecommerce-platform

```

### 2. Install dependencies

```bash
npm install

```

### 3. Available Scripts

In the project directory, you can run the following scripts:

| Command | Description |
| --- | --- |
| `npm run dev` | Runs the app in development mode at `http://localhost:5173`. |
| `npm run build` | Builds the app for production into the `dist` folder. |
| `npm run lint` | Lints the codebase using ESLint to catch errors and style issues. |
| `npm run preview` | Locally previews the production build generated by the build script. |

## 📁 Project Structure Highlights

* `index.html` - The application entry point shell.
* `eslint.config.js` - Global linting configuration using the modern Flat Config format.
* `src/main.jsx` - Main application bootstrap script.
